outputCW

  • outputCW(codes: number[]): Promise<void>
    • Defined in USBIF4CW_Device.ts:71

    Code0からCode6で与えられた文字列を表すコードに基づき、CW符号生成・自動出力を行います。

    0x05 CW符号出力 (Ver.2)

    Byte0Byte1Byte2Byte3Byte4Byte5Byte6Byte7
    0x05 Code0 Code1 Code2 Code3 Code4 Code5 Code6
    • 文字間スペース(初期値では3短点分の長さ)を自動的に出力します。
    • 1度のコマンド送信で最大7文字の出力予約ができますが、 8バイトのバッファに収まるように送信してください。
    • このコマンドを送信すると、内部モードがCW符号生成モードに切り替わります。
    • パラメータとして、CW符号の短点に対応するビットを0、 長点に対応するビットを1とする、独自のコード体系を使用します。 詳細はUSBIF4CW/Firmware/NativeCodeを参照してください。

    Parameters

    • codes: number[]

      文字を表す独自コード(最大7文字分)。 例: [0b00001000, 0b01000000, 0b01001000, 0b01001000, 0b11110000] (Hello)

    Returns Promise<void>

outputFSK

  • outputFSK(codes: number[]): Promise<void>
    • Defined in USBIF4CW_Device.ts:123

    Code0からCode6で与えられたボドーコードに基づき、 FSK符号生成・自動出力を行います。

    0x06 FSK符号出力 (Ver.2)

    Byte0Byte1Byte2Byte3Byte4Byte5Byte6Byte7
    0x06 Code0 Code1 Code2 Code3 Code4 Code5 Code6
    • 出力符号のパラメータは次のとおりです。
      • ボーレート: 45.45ボー
      • スタートビット: 1ビット
      • ストップビット: 1.5ビット
    • 1度のコマンド送信で最大7文字の出力予約ができますが、 8バイトのバッファに収まるように送信してください。
    • このコマンドを送信すると、内部モードがFSK符号生成モードに切り替わります。 リセットを行うか他のモードへの切り替えコマンドを送信するまで、 パドルやボタンの処理・CW符号生成処理は無効になります。

    Parameters

    • codes: number[]

      文字を表す独自コード(最大7文字分)。 例: [0b00001000, 0b01000000, 0b01001000, 0b01001000, 0b11110000] (Hello)

    Returns Promise<void>
